## Overview

Hyperliquid is a high-performance decentralized perpetuals exchange built on its own L1 blockchain. It provides deep liquidity through HyperCore precompiles and enables on-chain perps trading with sub-second finality. Our integration leverages Hyperliquid's native builders code to deliver institutional-grade trading infrastructure through accessible interfaces.


## Technical Details

Hyperliquid uses a custom consensus algorithm called HyperBFT inspired by Hotstuff. The platform supports:

- **200k orders/second** throughput
- **One-block finality** for all operations
- **Fully on-chain** order books
- **Transparent liquidations**

Learn more about Hyperliquid: [Hyperliquid Documentation](https://hyperliquid.gitbook.io/hyperliquid-docs)
